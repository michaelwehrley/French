Determiners
* Articles: a, an, the
* Possessives: my, your, his, her, its, ours, their, whose
* Demonstratives: this, that, these, those
* Numerals: one, two, three,...
* Ordinals: first, second, third...
* Quantifiers: a few, another, several, much, many, a lot of, any, some
